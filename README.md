# data-science-project-3
Toxic Comment Classification is a Natural Language Processing (NLP) project that aims to detect and classify harmful or abusive language in online text. The model is trained on user comments and predicts whether a comment is toxic, insulting, threatening, or generally safe.
# Objective

To automatically detect toxic content and help in moderating online discussions to maintain a safe environment.

⚙️ Technologies Used
Python
Pandas
NumPy
Scikit-learn
Natural Language Processing (NLP)
🧠 Model Used
Logistic Regression / Naive Bayes (based on implementation)
TF-IDF / CountVectorizer for text representation

# Overview

Fake News Detection is a machine learning project that classifies news articles as REAL or FAKE based on their textual content.
